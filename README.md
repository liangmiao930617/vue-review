# vue-review
独立开发一个vue项目

进入现在的公司后，公司所有开发的项目都是用vue，之前架构方面都是老大弄的，为了证明我也可以，我按照自己的方法在重新弄了一下。下面是我的经验总结：

一、搭建VUE脚手架： https://github.com/vuejs-templates/webpack 
$ npm install -g vue-cli
$ vue init webpack my-project
$ cd my-project
$ npm install
$ npm run dev

二、平台的UI框架是element-ui,之前平台是用的element-ui的1.3.7的版本，这个版本太老了，所以我选的是2.3.9：http://element-cn.eleme.io/2.3/#/zh-CN/component/quickstart
